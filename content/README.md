# MonopolyBattle - Content Directory

All Vietnamese game content and educational definitions live in this directory. 

## Structure
- `/narrator/`: Templates for the AI narrator based on decisions, events, monopoly checks, and context.
- `/quizzes/`: Quiz packs for rounds 3, 5, and 7 testing political economy concepts.
- `/concepts/`: Definitions and glossary terms.
- `/game/`: Economic decisions and events parameters.
- `/ui/`: User-customizable UI strings.

All files are formatted in standard UTF-8 JSON.
